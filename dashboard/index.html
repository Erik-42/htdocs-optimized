XAMPP Apache + MariaDB + PHP + Perl

Bienvenue dans XAMPP pour Linux 8.2.4

Vous avez installé avec succès XAMPP sur ce système ! Vous pouvez maintenant commencer à utiliser Apache, MariaDB, PHP et d'autres composants. Vous pouvez trouver plus d'informations dans la section FAQs section or check the HOW-TO Guides pour démarrer avec les applications PHP.

XAMPP est destiné uniquement à des fins de développement. Il dispose de certains paramètres de configuration qui facilitent le développement local, mais qui ne sont pas sécurisés si vous souhaitez que votre installation soit accessible à d'autres.

Démarrez le panneau de configuration XAMPP pour vérifier l’état du serveur.
